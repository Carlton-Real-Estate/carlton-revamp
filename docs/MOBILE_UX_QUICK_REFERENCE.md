## üîß CSS Snippets

### Bedrooms Centering
```css
/* Tablet */
@media (max-width: 1024px) {
    .search-fields .form-group:nth-child(4) {
        grid-column: 1 / -1;  /* Span full width */
        max-width: 400px;      /* Constrain width */
        margin: 0 auto;        /* Center it */
    }
}

/* Mobile */
@media (max-width: 768px) {
    .search-fields .form-group:last-child {
        width: 100%;
        margin: 0 auto;
    }
}
```
